# Installation

1. Open your GitHub profile repository: `dhanush229-lab/dhanush229-lab`.
2. Upload the complete `assets` folder.
3. Upload `.github/workflows/snake.yml` with the same folder structure.
4. Replace the existing root `README.md` with the new `README.md`.
5. Commit the files to the `main` branch.
6. Open the **Actions** tab and run **Generate contribution snake** once using **Run workflow**.
7. Wait for the workflow to create the `output` branch, then refresh your profile.

Do not upload only the README. The custom design depends on the SVG files inside `assets/`.
